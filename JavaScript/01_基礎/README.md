# JavaScript基礎編

JavaScriptの基本となる重要な概念を学びます。モダンな開発に必要な基礎知識を身につけましょう。

## 📚 この章で学ぶこと

### 1. [Script読み込み（defer/async）](./01_script-loading.md)

JavaScriptファイルの読み込み方法とパフォーマンスへの影響を学びます。

**学習内容：**
- scriptタグの基本
- defer属性の使い方
- async属性の使い方
- それぞれの違いと使い分け

**重要度：⭐⭐⭐⭐⭐**
ページ読み込みパフォーマンスに直結する重要なトピックです。

---

### 2. [モジュール（type="module"）](./02_modules.md)

JavaScriptのモジュールシステムとスコープ管理を学びます。

**学習内容：**
- type="module"の基本
- スコープの隔離
- グローバル変数の衝突を防ぐ方法
- import/exportの基礎

**重要度：⭐⭐⭐⭐**
大規模なアプリケーション開発には必須の知識です。

---

### 3. [ES6基礎知識](./03_es6-basics.md)

ES6（ES2015）で導入された主要な機能を学びます。

**学習内容：**
- let/constによる変数宣言
- アロー関数
- クラス構文
- テンプレート文字列
- 分割代入
- スプレッド構文
- for...ofループ
- Promiseの基礎

**重要度：⭐⭐⭐⭐⭐**
現代のJavaScript開発の基礎となる必須知識です。

## 🎯 学習目標

この章を完了すると、以下ができるようになります：

- ✅ scriptタグを適切に配置し、パフォーマンスを最適化できる
- ✅ defer/asyncの違いを理解し、使い分けられる
- ✅ モジュールを使ってスコープを管理できる
- ✅ ES6の主要な構文を使いこなせる
- ✅ let/constを適切に使い分けられる
- ✅ アロー関数を書ける
- ✅ 分割代入やスプレッド構文を使える

## 📝 学習の進め方

### ステップ1: 順番に読む

1. Script読み込み
2. モジュール
3. ES6基礎知識

の順で学習することを推奨します。

### ステップ2: コードを書く

各トピックのサンプルコードを実際に書いて、動かしてみましょう。

### ステップ3: 実験する

サンプルコードを少し変更して、動作を確認してみましょう。エラーが出たら、なぜエラーになるのか考えてみることも大切です。

## 💡 学習のヒント

### 初学者の方へ

- **焦らず一つずつ**: すべてを一度に理解しようとせず、一つのトピックをしっかり理解してから次に進みましょう
- **手を動かす**: 読むだけでなく、必ず自分でコードを書いて実行しましょう
- **エラーを恐れない**: エラーは学習のチャンス。エラーメッセージを読んで理解する練習をしましょう

### 効果的な学習方法

1. **サンプルコードを写経する** - まずは正確に写して動かす
2. **少しずつ変更する** - 一部を変えて動作を確認
3. **自分で書いてみる** - 何も見ずに同じものを書いてみる
4. **応用する** - 学んだことを使って小さなプログラムを作る

### 簡単なHTMLテンプレート

各トピックを試すための基本テンプレートです：

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>JavaScript学習</title>
</head>
<body>
  <h1>JavaScript学習</h1>

  <!-- ここにコンテンツを追加 -->

  <script>
    // ここにJavaScriptを書く
    console.log('Hello, JavaScript!');
  </script>
</body>
</html>
```

### 開発者ツールの使い方

ブラウザの開発者ツールを開く方法：
- Windows/Linux: `F12` または `Ctrl + Shift + I`
- Mac: `Cmd + Option + I`

**Console タブ**で `console.log()` の出力を確認できます。
