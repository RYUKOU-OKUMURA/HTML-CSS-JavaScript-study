Details開閉アニメーション（JavaScript版）

対応ブラウザ: すべてのモダンブラウザ（IE11を除く）
アニメーションの仕組み

Web Animations APIを使用して、滑らかな開閉アニメーションを実装しています。

  • 開く時: 高さ0から実際の高さへ
  • 閉じる時: 実際の高さから0へ
  • 透明度も同時にアニメーション
実装のポイント

重要な注意点

  1. 直下の要素には上下paddingを指定しない

    .details-contentにpaddingを指定すると高さの計算が狂います。

  2. 内側要素でpaddingを設定

    .details-innerでpaddingを指定します。

  3. scrollHeightを使用

    コンテンツの実際の高さを取得するために使用します。
