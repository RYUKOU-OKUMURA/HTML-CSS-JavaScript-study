2. レスポンシブグリッド（auto-fit + minmax）

ポイント: repeat(auto-fit, minmax(250px, 1fr)) で画面幅に応じて自動的にカラム数が変わる（ウィンドウサイズを変更してみてください）

カード1

レスポンシブに対応したカードレイアウト。画面幅に応じて自動的にカラム数が調整されます。

カード2

minmaxを使用することで、最小幅を保証しながら柔軟にレイアウトできます。

カード3

メディアクエリを書かなくても、自動的にレスポンシブになるのが特徴です。

.grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 24px; }

ポイント: grid-template-areas でレイアウトを視覚的に定義。レスポンシブ対応も簡単（画面幅768px以下でレイアウトが変わります）

.layout { display: grid; grid-template-areas: "header header header" "sidebar main main" "footer footer footer"; grid-template-columns: 200px 1fr 1fr; gap: 15px; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } .footer { grid-area: footer; }

4. Subgrid（要素の高さ自動揃え）

ポイント: grid-template-rows: subgrid で親グリッドの行を継承。各カードの見出し、画像、説明文、タグがそれぞれ同じ高さに自動調整されます

短いタイトル

これは短い説明文です。Subgridにより、他のカードの説明文の高さに自動的に合わせられます。

Web開発

これはもっと長いタイトルの例です

説明文の長さが異なっても大丈夫です。Subgridが自動的に高さを揃えてくれます。JavaScriptは一切不要で、CSSだけで実現できます。

CSS Grid

中程度のタイトル

ウィンドウサイズを変更しても、常に同じ行のカードの高さが揃います。レスポンシブに対応しています。

Subgrid
.container { display: grid; grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)); gap: 30px; } .card { display: grid; grid-template-rows: subgrid; /* ← ここが重要 */ grid-row: span 4; /* 親グリッドの4行分 */ row-gap: 15px; }

5. place-content: center（完全中央配置）

ポイント: place-content: center でグリッド内のコンテンツを縦横中央に配置

完全中央配置

Grid + place-content で簡単に実現

.center { display: grid; place-content: center; min-height: 100vh; /* 画面全体の高さ */ }
