# CSS学習ガイド

現代のWeb開発に必須のCSSテクニックを6つのカテゴリーに分けて解説しています。

## 📚 学習の進め方

このガイドは以下の順序で学習することを推奨します：

1. **セレクタ編** - モダンなセレクタの使い方
2. **レイアウト編** - Grid Layout と Subgrid
3. **視覚効果編** - filter、blend-mode、clip-path
4. **モダンプロパティ編** - clamp()、ビューポートユニット
5. **アクセシビリティ編** - ユーザー配慮の設計
6. **実装パターン編** - 実践的なパターン集

## 📖 目次

### 1. セレクタ編

モダンなセレクタを使いこなして、効率的にスタイルを適用する方法を学びます。

- [セレクタ編の詳細](./01_セレクタ/README.md)

**学習内容:**
- `:has()` - 親セレクタとして使える画期的なセレクタ
- `:is()` - セレクタをまとめて簡潔に記述
- `:where()` - 詳細度0で上書きしやすいスタイル
- セレクタの詳細度（Specificity）の理解

**実装サンプル:**
- [`:has()` セレクタのデモ](./01_セレクタ/examples/has-selector-demo.html)
- [`:is()` と `:where()` のデモ](./01_セレクタ/examples/is-where-demo.html)

---

### 2. レイアウト編

CSS Grid Layout を使った2次元レイアウトの構築方法を学びます。

- [レイアウト編の詳細](./02_レイアウト/README.md)

**学習内容:**
- Grid Layout - 2次元レイアウトの基礎から応用
- Subgrid - 複数要素の高さを自動的に揃える
- gap - アイテム間の余白を簡単に設定
- 配置プロパティ - inset、margin-inline、place-content

**実装サンプル:**
- [Grid Subgrid のデモ](./02_レイアウト/examples/grid-subgrid-demo.html)

---

### 3. 視覚効果編

フィルター効果やブレンドモードを使った視覚表現を学びます。

- [視覚効果編の詳細](./03_視覚効果/README.md)

**学習内容:**
- filter - ぼかし、明度、グレースケールなどの視覚効果
- mix-blend-mode - Photoshopのようなブレンドモード
- clip-path - 要素を図形で切り抜く
- transform独立プロパティ - translate、scale、rotateを個別制御

---

### 4. モダンプロパティ編

レスポンシブ対応やメンテナンス性を向上させるモダンなプロパティを学びます。

- [モダンプロパティ編の詳細](./04_モダンプロパティ/README.md)

**学習内容:**
- currentColor - 色の同期を簡単に
- clamp() - メディアクエリ不要のレスポンシブサイズ
- 新ビューポートユニット - svh、dvh、lvh（モバイル対応）
- word-break - 正しいテキスト折り返し

---

### 5. アクセシビリティ編

すべてのユーザーが快適に利用できるWebサイトを作るための技術を学びます。

- [アクセシビリティ編の詳細](./05_アクセシビリティ/README.md)

**学習内容:**
- モダンメディアクエリ - Range Syntaxで簡潔に
- any-hover - 入力方式に応じたホバー対応
- prefers-reduced-motion - アニメーション削減設定への対応
- Visually Hidden - スクリーンリーダー対応

---

### 6. 実装パターン編

実践で使える具体的な実装パターンを学びます。

- [実装パターン編の詳細](./06_実装パターン/README.md)

**学習内容:**
- アコーディオン（Details要素のアニメーション）
- タブUI（アクセシブルな実装）
- カードレイアウト（Grid + Subgrid）
- レスポンシブ画像ギャラリー
- スムーススクロール

**実装サンプル:**
- [Details要素のアニメーション](./06_実装パターン/examples/details-animation-js.html)
- [アクセシブルなタブUI](./06_実装パターン/examples/accessible-tabs.html)

## 🎯 学習目標

このガイドを完了すると、以下のスキルが身につきます：

- ✅ 実践的な実装パターンを適用できる
